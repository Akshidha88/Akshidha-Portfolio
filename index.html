I developed it as part of my college mini-project, combining backend logic, AI model integration, and user experience design to create a smart assistant that responds to both text and context-aware queries.

I handled everything from backend APIs to dynamic frontend UI, ensuring a smooth user experience and responsive design. This project helped me strengthen my full-stack development skills.

The dashboard visualizes key metrics like grades, attendance, and performance trends with interactive charts and filters. This project deepened my understanding of data visualization, storytelling through data, and decision-making insights using BI tools.
